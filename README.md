# UPS Customer System

A customer management dashboard for UPS, integrated with Google Sheets via Google Apps Script (GAS).

## Features
- **Customer Registration**: Register new customers with vehicle details.
- **Customer List**: View and filter customers.
- **Approach History**: Track interaction history (calls, visits) with two-way synchronization to Google Sheets.
- **Premium UI**: Modern, responsive dashboard design.
