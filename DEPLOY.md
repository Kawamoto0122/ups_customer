# Deployment Guide

## GitHub Configuration
The local repository has been configured with the remote:
`https://github.com/Kawamoto0122/ups_customer.git`

To push your code, run:
```bash
git push -u origin main
```

## Vercel Deployment
1.  Log in to [Vercel](https://vercel.com).
2.  **Add New...** > **Project** > **Import from GitHub**.
3.  Select `Kawamoto0122/ups_customer`.
4.  Click **Deploy**.
